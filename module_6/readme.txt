
Задача - предсказание цены авто по тестовому набору данных

Исходные данные:
    в задание присутствует датасеты train и test

Задачи решаемые в проекте:
    1. Парсинг данных с сайта auto.ru, обновление данных
    2. Приведение разных датасетов (train и test и спрарсенный с сайта - train_new), в единеое пространство признков.
    3. Анализ данных признаков, очитска , приведеление к одному типу, удалание высококоррелированнызх признаков
    5. Обучение различными алгоритмами
    6. Поиск оптимального алгоритма

Файлы
    parser.py - парсер данных с сайта auto.ru
    price-prediction-v16 - плейбук с решеним и графиками анализа данных
    index.py - рабочий  скрипт написания обработки данных
